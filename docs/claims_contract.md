# Claims Contract

| Allowed claims | Disallowed claims | Required qualifiers |
|---|---|---|
| "CCI reports internal temporal dependence under conditioning." | "CCI proves causation." | Specify estimator assumptions and conditioning context. |
| "Bias-corrected CCI is used for cross-architecture comparison." | "Raw CCI alone establishes autonomy." | Report raw + corrected and note finite-sample inflation risk. |
| "`1.0 bit` is an operational threshold in this setup." | "`1.0 bit` is a universal autonomy threshold." | Mark threshold as setup-specific, not universal. |
| "Transformer non-zero CCI indicates conditional dependence signal." | "Transformer CCI proves agentic subjecthood." | Include explicit non-claim language from `CLAIMS.md`. |
| "Interpretive protocols are vision-layer references." | "Interpretive protocols are empirical proof." | Separate interpretive layer from benchmark evidence. |
